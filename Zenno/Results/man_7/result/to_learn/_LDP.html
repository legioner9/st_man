    ИМЯ

    LDP - Введение в Проект Документации Linux (Linux Documentation Project), помощь, руководства и документы  

    ОБЗОР

    Проект Документации Linux (сокращеное английское название LDP) предоставляет сообществу Linux разные виды свободных и бесплатных ресурсов документации, включая руководства, FAQ (список часто задаваемых вопросов), HOWTO и страницы руководства man.

     

    АВТОРЫ

    Различные документы в архивах LDP сопровождаются отдельными авторами, которые указываются в начале каждого документа HOWTO. Если у вас есть какие-либо вопросы или дополнения к документу, мы призываем вас напрямую связаться с авторами.

     

    WEB-СТРАНИЦЫ

    LDP имеет свой собтвенный web-сайт и поддерживает множество разных проектов переводов, которые связаны с главным web-сайтом LDP, адрес которого
    http://www.tldp.org/
    вы, возможно, захотите добавить к своим загладкам.

     

    СТРАНИЦЫ РУКОВОДСТВА MAN

    Существует web-страница с информацией о состоянии и переводах страниц руководства man по адресу
    http://www.win.tue.nl/~aeb/linux/man/

     

    СПИСКИ РАССЫЛКИ

    LDP имеет несколько списков рассылки, таких как

    <announce@en.tlpd.org> Анонсы проекта LDP
    <discuss@en.tldp.org>
    Общая дискуссия по проекту LDP
    <docbook@en.tldp.org>
    Вопросы об использовании DocBook

    Информацию о том, как подписаться на эти списки рассылки, смотрите на указаннм выше сайте.

    Если вы интересуетесь как написать ваш документ для проекта LDP с помощью DocBook, то, верояно, вы захотите также подписаться на один из списков рассылки OASIS DocBook. Информацию о том как это сделать, смотрите по адресу http://docbook.org/mailinglist/index.html.

     

    ФАЙЛЫ

    Большинство дистрибутивов включают в установку документы HOWTO и mini-HOWTO
    /usr/doc/ (старое место для документации)
    /usr/share/doc/ (новое место для документации)
    /usr/share/doc/HOWTO/ (файлы HOWTO)
    /usr/share/doc/HOWTO/mini/ (файлы mini-HOWTO)
     

    СМОТРИ ТАКЖЕ

    man(1), xman(1x), info(1)

    страницы info, которые можно прочитать с помощью emacs(1) или info(1)
